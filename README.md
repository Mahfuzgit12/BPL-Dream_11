# 🏏 BPL Dream 11 – Fantasy Cricket Team Builder

BPL Dream 11 is a fantasy cricket web application inspired by popular fantasy gaming platforms like Dream11 and MPL.  
The project was built with the idea of creating a simple but interactive experience where users can build their own cricket team by selecting players within a limited budget.

This project mainly focuses on frontend development, responsive UI design, component-based architecture, and smooth user interaction using modern React technologies.

---

# 📖 About The Project

The application allows users to explore a list of cricket players and create their own fantasy squad.  
Every player has a specific role, rating, and price, so users need to make smart decisions while building their team.

The interface was designed to feel clean, modern, and easy to use while maintaining a fantasy sports theme.

This project was also a great opportunity to practice:
- React component structure
- State management
- Dynamic rendering
- Responsive layouts
- Modern UI/UX design

---

# ✨ Features

### 🧑‍🤝‍🧑 Player Selection System
Users can browse through available players and add them to their fantasy team.

### 💰 Budget Management
Each player has a price, and users start with a limited amount of coins.  
This creates a more realistic fantasy team building experience.

### 📋 Player Information
Every player card includes:
- Player image
- Country
- Role
- Batting style
- Bowling style
- Rating
- Price

### 🔄 Team Management
Users can:
- View selected players
- Remove players from the squad
- Track remaining balance
- Monitor total selected players

### 📱 Responsive Design
The website is fully responsive and works smoothly on:
- Desktop
- Tablet
- Mobile devices

### 🎨 Modern User Interface
The UI includes:
- Clean layouts
- Modern card design
- Hover effects
- Gradient buttons
- Smooth spacing and alignment

# ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone <repository-link>
```

### Go to the Project Folder

```bash
cd bpl-dream-11
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The project will run locally on:

```bash
http://localhost:5173
```

# 🚀 How The Application Works

1. Users can view all available cricket players.
2. Every player has their own price and role.
3. Users can select players to build a fantasy team.
4. Coins decrease automatically after selecting players.
5. Users can switch between:
   - Available Players
   - Selected Players
6. Selected players can also be removed from the team.
